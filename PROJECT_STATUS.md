## 📖 專案程式功能概述

AI PRO HUB 是一個提供 AI 專案監控與管理的單頁式應用程式 (SPA)。我們透過折疊式的專案卡片、微型圖表，和直觀的主題切換功能，為開發人員與管理員展示多個 AI 開發專案的最新狀態。前端使用 React 19、TypeScript 和 Tailwind CSS 打造優美的玻璃擬態 (Glassmorphism) 排版與動畫體驗。
